Getting data

Queries

All queries are passed as vectors of pairs. For example, filtering interactions that are of the mutualist type can be done with [Pair("type", "mutualism")].

Mangal.datasets — Function.
datasets()

When called with no arguments, this function will return a list of the most recent datasets. The results will be returned as a vector of MangalDataset object.
